Connect your AI in 3 steps

Everything runs on your own API key. Your data never leaves your machine.

1

Deploy Cloudflare Worker proxy

Go to workers.cloudflare.com → Create Worker → paste worker.js → Deploy. Copy the URL.

2

Get your AI API key

Gemini: aistudio.google.com → Create API Key. Free tier included.

3

Connect & Test

Paste your key and proxy URL into the test card. Green tick means you're live and ready to launch.
